### Opengl GUI ###

__about__
This is a demo to see how well opengl functions as a method for displaying the results of opencv.
There is a slider that will transition from one image to another.

__controls__
<w,a,s,d> - translate image
<left mb> - translate image
<-,+/=>   - zoom in and out

__compiling__
make

__running__
./GUI [image 1] [image 2]
make sure both images are the same size.
